# hw-04 — Лабораторна робота 4

Проєкт виконано для лабораторної роботи 4: блокова модель CSS, відступи, рамки, позиціонування та Flexbox.

Використано попередній проєкт з ЛР1–ЛР3. Додано `box-sizing: border-box`, пропрацьовано `margin`, `padding`, `border`, створено адаптивні flex-контейнери, закріплено навігаційне меню зверху сторінки, використано різні типи позиціонування (`sticky`, `relative`, `absolute`, `fixed`), збережено сторінку таблиці для варіанта №5.
